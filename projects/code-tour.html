SmartLabeling Review Route

FastAPI routers

  • SmartLabeling `core/routers`
  • Architecture flow
03 · Model and artifact logic

Mask, CLIP, project, and export services

Inspect the services to see how model-facing logic and export artifacts are separated from the desktop UI.
