## Recommended Features

1. **Positioning-first homepage hero** - the current first screen still says "personal blog" while stronger portfolio proof begins lower on the page.
2. **Visitor intent router** - compact routes for recruiter, technical reviewer, reader, and collaborator make the page easier to navigate by intent.
3. **Project evidence receipts / Project TL;DR cards** - highest technical-showcase value among low-cost options.
4. **Proof narrative ladder / Proof Map compression** - improves content presentation without adding unrelated sections.

## Eliminated / Deferred

- WebGL and shader motifs: interesting but decorative before the homepage story is clear.
- Capability filters / reviewer mode that hides content: too much state and maintenance for current content volume.
- Homepage changelog: stale-date risk.
- Archive mood filters and search result reasons: lower immediate product value than first-visit comprehension.

## Product Decision

Ship a single **homepage evidence-routing pass**: positioning hero, Start Here routes, compressed proof ladder, and project evidence receipts.
