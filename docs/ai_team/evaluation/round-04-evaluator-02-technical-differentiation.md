## Recommended Features

### 1. Project Evidence Receipts + Runtime Honesty Badges

Best technical-showcase feature. It turns project cards into clearer proof surfaces with `Claim`, `Evidence`, and `Where to inspect`.

### 2. First-Viewport Identity Panel + Start Here Routes

Highest value for first-time comprehension. It should link to existing anchors/pages and avoid hidden content or filtering.

### 3. Proof Map Visual Compression + Proof-to-Project Bridge

Improves content presentation while preserving the existing evidence-first structure.

### 4. Static Site Pipeline Lens

Strong future technical showcase because it makes the website itself part of the portfolio, but it should stay compact.

## Eliminated / Deferred

- Mobile section jump bar: useful, but secondary to hierarchy.
- Search result reasons and query chips: touches existing search behavior and is not the current bottleneck.
- Curated archive/image rail: warmer, but weaker technical showcase.
- WebGL motif: too decorative for this round.
- 60-second review mode/capability filtering: unnecessary state and maintenance burden.
