## Recommended Features

1. **First-Viewport Identity / Recruiter Start Here** - replace the generic blog opening with a concise professional thesis.
2. **Project TL;DR / Evidence Receipts** - reframe cards around `Problem`, `Role`, `Evidence`, and `Where to inspect`.
3. **Role-to-Proof Matrix / Proof Map Compression** - shorter labels, clearer ownership, direct evidence links.
4. **Visitor Intent Router** - compact static route block.
5. **Contact Intent Panel** - useful later after the route/proof path is stronger.

## Eliminated / Deferred

- WebGL, shader headband, proof constellation: decorative for the current problem.
- Capability filters and 60-second review mode: over-engineered for current content volume.
- Archive rails and mood filters: personal warmth, but not recruiter clarity.
- Public changelog: stale-date risk.
- Search upgrades: lower recruiter conversion impact.
