Weights: value 40%, technical display 25%, implementation cost 20%, long-term return 15%.

| Rank | Feature | Score | Decision |

## Recommended

- Project Case Studies for SmartLabeling and Yomii.
- Hiring Snapshot focused on “where to verify me.”
- Skill Evidence Matrix with skill -> proof -> project/code links.
- Code Reading Guide inside project pages.
- Site as Project / Verified Outcomes.
- SmartLabeling Visualizer embedded into the SmartLabeling case study.

## Eliminated

- Compute particle identity / 3D timeline: high visual, low evidence.
- Knowledge graph / archive constellation: sparse-data and low information-density risk.
- WebGPU AI Playground / Mini Vision Lab / GPU Image Bench: high scope and maintenance.
- Visitor Funnel Panel: low recruiter signal and privacy overhead.
- XHS Photo Notes: weak technical hiring relevance.
- Engineering Flashcards: too close to generic interview prep.
