## Recommended

- Project Case Studies / Cards
- Hiring Snapshot
- Capability Evidence Matrix
- SmartLabeling case study with static visuals
- Curated Reading Paths
- Concise Site Build Log

## Eliminated / Deprioritized

- WebGPU graphics lab and 3D/particle identity: performance, fallback, motion, and mobile risk.
- Knowledge graph / archive constellation / AI skill graph: low scannability and likely clutter.
- WebGPU AI playground / GPU image bench / mini vision lab: scope-heavy.
- Visitor funnel panel: privacy and analytics complexity.
- Command palette: defer because focus management and accessibility must be excellent.
