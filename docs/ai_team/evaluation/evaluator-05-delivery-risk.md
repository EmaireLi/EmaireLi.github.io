# Evaluator 5: Delivery And Risk Critic

Weights: value 40%, technical display 25%, implementation cost 20%, long-term return 15%.

## Ranking

| Rank | Feature | Score | Reason |
|---:|---|---:|---|
| 1 | Project Case Study Pages / Cards | 87 | Highest value, safe plain HTML/CSS/JS implementation. |
| 2 | Hiring Snapshot | 81 | Low-cost conversion improvement. |
| 3 | Capability / Skill Evidence Matrix | 80 | Credible if linked to real evidence. |
| 4 | Build Log / Verified Outcomes / Colophon | 78 | Turns existing site systems into portfolio evidence. |
| 5 | Explainable Local Search | 78 | Static-site-friendly technical enhancement. |
| 6 | Canvas-first Graphics Lab | 77 | Good direction, but should start Canvas/WebGL-first. |
| 7 | Static Portfolio Snapshots | 74 | Strong proof if image quality and size are controlled. |
| 8 | Learning Route / Milestone Timeline | 70 | Useful if tied to real artifacts. |

## Recommended

- Project Case Study Pages / Cards
- Hiring Snapshot + Capability Evidence Matrix
- Build Log / Site as Project
- Explainable Local Search
- Canvas-first Graphics Lab as a later phase

## Eliminated

- WebGPU-first playground: compatibility and QA cost too high for first round.
- Visitor funnel panel: privacy and Worker expansion risk.
- 3D timeline / particle identity: visual risk and motion concerns.
- Knowledge graph: sparse data and maintenance cost.
- Mini Vision / GPU image bench: likely to become half-finished.
- Local publishing assistant: generated summaries risk content trust.
- Now/changelog: staleness risk.
- XHS Photo Notes: weak hiring/technical signal.
