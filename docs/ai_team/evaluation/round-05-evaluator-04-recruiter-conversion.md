## Recommended Features

1. Personal Positioning Microcopy Pass
2. Contact Intent Panel
3. Reading Route Cards / Featured Reading Path
4. Evidence Freshness Markers
5. Curated Archive Lanes / Windows

## Conditional / Secondary Features

- Site Pipeline Lens is worthwhile if presented as "how this site is maintained" rather than as a separate external engineering project. It has good technical showcase value but should stay compact.
- Search Match Reasons / Context is technically strong, but likely too much complexity unless the existing search/discovery experience is already important.
- Account Identity Map can help trust, but it should be restrained.

## Eliminated Features

- Project Constraint Cards: too centered on external GitHub/project presentation instead of improving the personal website itself.
- AI Boundary Ledger: low recruiter-conversion value unless AI usage is a core part of the site identity.
- Decision Receipts Strip: too abstract for most recruiters.
- Homepage Review Minimap: limited value unless the homepage is very long.
- Visual Archive Preview Strip: useful only if visual content is central.
