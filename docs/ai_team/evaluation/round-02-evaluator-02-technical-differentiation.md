1. SmartLabeling Mask Visualizer
2. Hiring Snapshot + Capability Evidence Matrix
3. Explainable Local Search
4. Site-as-Project / Engineering Colophon

## Eliminated

- Standalone WebGPU / Shader / Engine Mini Lab
- GPU Capability Inspector
- Metadata Assistant for Posts
- Archive Metrics Dashboard / Personal Metrics Strip
- XHS Photo Notes Gallery
- Recruiter FAQ / Contact Intent Guide
