## Recommended Features

1. Site Pipeline Lens
2. Curated Archive Lanes / Windows
3. Contact Intent Panel

Optional companion: Personal Positioning Microcopy Pass, because it can tighten labels and reduce cognitive load while the above features are added.

## Eliminated / Deprioritized

- Role Readiness Scorecard: too self-rating-heavy.
- Project Constraint Cards: too focused on external/project case-study evaluation for this corrected round.
- Visual Archive Preview Strip: higher accessibility and layout risk than archive lanes.
- Homepage Review Minimap: premature until the homepage structure settles.
- AI Boundary Ledger: only worth a small note if tied to site authorship/process.
