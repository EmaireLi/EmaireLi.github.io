## Recommended

1. Curated Reading Paths / First Reads / Reading Lanes
2. Contact Intent Cards/Panel
3. Account Identity Map as optional follow-up

## Eliminated / Deferred

- Site Pipeline Playback
- Proof Snapshot Strip / Recruiter Snapshot Bar / Guided Start Here Visitor Path
- Visual Archive Preview Strip
- Personal Operating System Page
- Project-to-Post Bridge unless framed around personal learning/writing
