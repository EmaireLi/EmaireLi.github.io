# Evaluator 1: Product Value Critic

Weights: value 40%, technical display 25%, implementation cost 20%, long-term return 15%.

## Ranking

| Rank | Feature | Score | Reason |
|---:|---|---:|---|
| 1 | Project Case Study Pages / Cards | 9.0 | Highest hiring value; turns shallow project links into proof of ownership, architecture, tradeoffs, and outcomes. |
| 2 | Hiring Snapshot / Hiring Signal Panel | 8.5 | Low cost, high recruiter value; gives a 10-30 second path to identity, direction, proof, and contact. |
| 3 | Capability / Skill Evidence Matrix | 8.2 | Valuable if evidence-based and linked to real work. |
| 4 | SmartLabeling Visualizer | 8.0 | Strong because it ties directly to an existing AI project and avoids paid APIs. |
| 5 | Explainable Local Search | 7.8 | Practical improvement plus technical demonstration. |
| 6 | Code Reading Guide / Verified Outcomes / Site Colophon | 7.4 | Helps interviewers inspect real code and makes the site itself evidence. |
| 7 | WebGPU / Graphics Lab | 7.2 | Strong technical showcase, but higher compatibility and cost risk. |
| 8 | Curated Reading Paths | 7.0 | Useful navigation, lower technical signal. |

## Recommended

- Project Case Study Pages
- Hiring Snapshot
- Skill Evidence Matrix
- SmartLabeling Visualizer
- Explainable Local Search

## Eliminated / Deprioritized

- Visitor funnel panel: privacy and backend complexity before project evidence exists.
- WebGPU AI Playground: broad scope and unclear personal-site value.
- Compute particle identity card: visual impact but weak proof.
- Engine roadmap 3D timeline: flashy self-narrative risk.
- XHS Photo Notes: good archive value but weak hiring/technical signal.
- Static resume page: ordinary; better after stronger proof exists.
