Site pipeline lens

这个网站本身也是一个可检查的作品。

从写作、导入、生成索引到搜索和发布，尽量用静态文件把内容系统做清楚。

    Write locally

  2. 02

    Import notes

    Inspect importer
  3. 03

    Build the manifest

    脚本扫描 posts/，生成归档和搜索共用的文章清单。

    View manifest
  4. 04

    Browse and search

    首页归档和本地搜索共享同一份数据，读者不用离开静态站点也能找到内容。

    Respond and publish
